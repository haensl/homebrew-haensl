# homebrew-haensl

MacOS formulae for the Homebrew package manager

## Quick start

1. Add formulae repo to brew

  ```bash
  brew tap haensl/haensl
  ```

2. Install stuff, e.g.

  ```bash
    brew install goto
  ```

## Available formulae

* [goto](https://github.com/haensl/goto)
